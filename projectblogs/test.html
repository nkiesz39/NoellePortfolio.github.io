Background

This project was undertaken as the final project for the IBM Data Analyst Certification.
In this project, it is assumed that we are a Data Analyst at a Global IT and Consulting Firm. In order to keep pace with changing technologies, the organization regularly analyzes data nad trends to identify future skills requirements.

As a Data Analyst, we have been tasked with collecting data from various sources and identifying trends for this year's report on emerging technology skills. The purpose of the results are to aid developers and employers in staying relevant in the job market and guide employer training and upskilling support.

If you would like to follow along or see the code for this project you can visit the repo or the Interactive Cognos Dashboard.
